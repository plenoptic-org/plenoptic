# CONTRIBUTING

`plenoptic` is a python library of tools to help researchers better understand their models. We welcome and encourage contributions from everyone!

By participating in this project, you are expected to uphold the [Code of Conduct](CODE_OF_CONDUCT.md).

For details on how to contribute, please view the [contributing page in our documentation](https://docs.plenoptic.org/docs/branch/main/developers/contributing.html).
